Structure de base

HTML ( hypertext Markup language ), c'est le contenue pour structurer une page web

les Balises sont des éléments HTML encadré de chevron (>)

les attributs sont les éléments ajoutés dans les Balises comme href style alt ....

un éléments est ce qui est entre une balise d'ouverture et de fermeture

Type de contenu HTML

TEXTE : contenu brut à afficher sur la page

MULTIMÉDIA : contenu comme image , videos ou son (balise img ou balise video)

FORMULAIRE : contenu interactif permettant la saisie d'information

LIEN : lien hypertext qui redirige vers une autre page ou section

CLASS : utiliser pour appliquer des styles CSS ou des script JAVASCRIPT

ID (id) identifiant unique d'un element
